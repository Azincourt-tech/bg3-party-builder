# ⚔️ BG3 Party Builder

Interactive Baldur's Gate 3 party build planner and guide.

## 🎮 Features

- 🏹 **Complete builds** for 4 characters (Tav, Lae'zel, Karlach, Shadowheart)
- 📊 **Stats, skills, spells** and equipment recommendations
- 📈 **Level progression** slider (1-12) with tips for each level
- 💡 **Combat tips** and combo strategies
- 🌙 **Dark/Light theme** toggle
- 📱 **Fully responsive** design

## 🛠️ Tech Stack

- HTML5
- CSS3 (CSS Variables, Grid, Flexbox)
- Vanilla JavaScript

## 🚀 Running locally

Just open `index.html` in your browser!

```bash
# Or with a local server
npx serve .
```

## 📦 Deploy

This project is designed to be hosted on **GitHub Pages**.
